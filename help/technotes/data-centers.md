# Customer Journey Analytics hosting locations

Adobe Customer Journey Analytics is hosted in enterprise-class data centers from public-cloud service providers in North America, Europe, and APAC.

Upon provisioning, customers designate the region in which their Adobe Experience Platform data will reside. The data ingested into Customer Journey Analytics from the Adobe Experience Platform data lake will be stored in the same region.

See [Regional data collection](https://experienceleague.adobe.com/en/docs/core-services/interface/data-collection/rdc) in Adobe CX Enterprise documentation for more information.


## View the data center where your data is stored

>[!NOTE]
>
>It is not possible to move data from one data center to another.

To view which data center your data is stored in:

1. Log in to [Adobe CX Enterprise](https://experience.adobe.com) using your Adobe ID credentials.

1. Select **[!UICONTROL Experience Platform]** from the app switcher ![App](/help/assets/icons/Apps.svg) at the top-right of the interface.

1. The region code for your assigned data center is shown in the upper-right portion of Experience Platform.

   ![Data center on Experience Platform home page](assets/experience-platform-data-center.png)

1. Use the following table to understand what geographical area your region code is associated with:

   | Adobe region code | Cloud provider | Geographic region |
   |-------------------|-------|-------------------------------------------|
   | VA7               | Azure | U.S. (default)  |
   | VA6               | AWS   | U.S. (by request) |
   | NLD2              | Azure | Netherlands, Amsterdam |
   | CAN2              | Azure | Canada Central, Toronto |
   | AUS5              | Azure | Australia, Sydney |
   | GBR9              | Azure | U.K., London    |
   | IND2              | Azure | India    |
   | CHE2              | Azure | Switzerland   |
